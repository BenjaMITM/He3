# He3

He3 est avant tout un navigateur flottant à fenêtres multiples.

! [he3_logo] (https://github.com/slashlos/He3/raw/master/he3_logo256.png "He3 fait Windows")

He3 - anciennement connu sous le nom de "Helium 3", prend en charge plusieurs fenêtres en tant que documents, chacune avec sa propre
les paramètres ainsi que les préférences globales telles que les listes de lecture et l'historique, et
possibilité d'affecter les fenêtres via un état du Finder, une icône de dock et des menus contextuels.

He3 vous permet de regarder des médias, de naviguer sur le Web et bien plus encore
vous restez productif. Votre contenu ne tombera jamais derrière les autres
Windows même lorsque vous changez de tâche.

He3 prend en charge un certain nombre de préférences par fenêtre telles que
- titrage à masquage automatique,
- mode de translucidité personnalisable
- taille et position

######

qui vous permet de voir simultanément votre contenu et votre travail, et de le restaurer, sans intercepter les clics de souris lorsqu'il n'est pas actif ou translucide. Vous avez la possibilité de cliquer, faire glisser, faire défiler derrière votre contenu flottant.

## Premiers pas - He3

Comme les versions précédentes, Windows améliorera votre multitâche, flottera au-dessus de votre travail, gardant les choses importantes en vue, mais c'est maintenant une application à fenêtres multiples, basée sur des documents, mais pas de «documents» à gérer. Tous les paramètres sont automatiques et internes et privés.

Toutes les fenêtres He3 sont des panneaux non activables, ils sont destinés à ne pas gêner votre travail. Ceci est particulièrement important lorsque vous décidez d'affecter la transparence d'une fenêtre. Lorsque vous souhaitez affecter une fenêtre, maintenez enfoncée la touche `SHIFT` pour la survoler - la rendant active, ou utilisez l'icône d'état du Finder si elle est disponible pour affecter la fenêtre principale de l'application. Enfin, lorsque vous souhaitez déplacer une fenêtre, utilisez sa barre de titre ou maintenez la touche `CMND` (`% `') enfoncée pour faire glisser le contenu. Si la fenêtre est translucide, vous devrez d'abord la rendre active par la méthode * shift into * mouse over.

Les paramètres sont conservés par document et restaurés puis rappelés. Les préférences par fenêtre peuvent également être affectées via l'élément de menu Préférences sous le menu Windows et le menu d'état du Finder He3 pour la fenêtre actuelle. Le menu Windows indiquera également quelle est la fenêtre actuelle.

Étant donné que la collaboration est essentielle, le partage de votre environnement est assuré en les exportant sous forme de listes de lecture et d'éléments de lecture individuels - c'est-à-dire un «document». Vous pouvez les glisser-déposer dans les fenêtres, les exporter vers le système et les recevoir d'autres personnes.

Enfin, quelques fonctionnalités plus sophistiquées:
- flotter au-dessus des applications plein écran comme Keynote ou Powerpoint
- partager des liens Web, etc. depuis Safari
- prend en charge les cookies d'authentification
- fenêtres illimitées, onglets
- Les listes de lecture `.hpl` et PlayItems` .hpi` sont des types de fichiers de partage d'exportation He3
- Les instantanés fournissent une copie visuelle instantanée de la fenêtre sous forme de fichiers `.png` vers votre destination
- ** Hide Away ** - appuyez sur `CNTL + OPTION + CMND` pour activer la visibilité de toutes les fenêtres

### Menus

Les menus He3 vous permettent d'affecter la fenêtre courante. Cela inclut les suspects habituels:
- menu d'application
- Menu d'icônes d'état du Finder
- menu d'icônes du dock

######

ainsi qu'un menu contextuel par fenêtre.

### Menu Documents

Pour le contenu basé sur des URL de fichiers plutôt que sur des sites Web, fenêtre de document He3,
lorsqu'une sauvegarde de fichier est possible, vous permettra de sauvegarder le document. Tous les fichiers
les opérations pour le magasin He3 aux préférences d'application par défaut de l'utilisateur,
qui sont globaux pour tous les documents, mais les paramètres individuels par document sont
capturé comme sa taille de cadre de fenêtre, son emplacement, son titre. Voir les listes de lecture ci-dessous.

### Listes de lecture

He3 gère deux types d'informations: l'historique et les listes de lecture.

Depuis toutes les fenêtres He3, un élément de menu contextuel, Playlists, accède à
toutes les informations enregistrées stockées dans le magasin par défaut des applications des utilisateurs
pour He3.

Par menu individuel, He3 maintient un historique des URL, ainsi que la capacité à héberger
plusieurs fenêtres liées à l'utilisateur sous forme d'onglets. La feuille Playlist - par fenêtre ou globale
Fenêtre (s) de liste de lecture, vous permettent de glisser-déposer pour organiser votre travail.

### Sauver votre peau

He3 enregistrera automatiquement l'état de toutes les fenêtres. Une liste d'historique, une réserve de pots de cookies sont également disponibles et les deux peuvent être effacés ou désactivés à volonté. Sauf si explicitement choisi, tout le stockage est interne, mais facilement transférable vers ou depuis les fichiers `.hpl` (playlist) et` .hpi` (playitem).

Les catalogues d'historique dans l'ordre, votre visualisation, peuvent être triés, exportés et importés, réattribués et aplatis pour agréger les doublons, en conservant également un décompte. Tout cela peut être géré ou désactivé.

Ce sont vos données, utilisez-les comme vous le souhaitez!
